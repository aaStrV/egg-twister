# egg-twister


#### eggs-twister is a simple periodic signal generator

Uses 16x2 display and 5-keys panel

Period = PhaseA + PhaseB

- Phase A - output level is low, minutes
- Phase B - output level is high, seconds/10

Screen:

`*Period` | `*Impulse`
--------- | ----------
`min:120` | `<ds:299>`
